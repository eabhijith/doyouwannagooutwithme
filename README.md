## Getting Started
1. Clone this repo 
```
git clone https://github.com/eabhijith/doyouwannagooutwithme.git
```
2. Direct to the root
```
cd /doyouwannagooutwithme
```
3. Deploy to your desired org
```
Make changes to the index.html 
```
